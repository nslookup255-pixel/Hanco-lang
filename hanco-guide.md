## 1. 한코란?

**한코(Hanco)** 는 한국어 키워드로 작성하는 프로그래밍 언어입니다. Python 위에서 동작하며, Lexer → Parser → AST → VM 파이프라인으로 실행됩니다.

### 특징

- 모든 키워드가 한국어
- `< ~>` 꺽쇠 기반 블록 문법
- `()` 함수 호출 문법
- 정적 자료형 변수 선언 (선언 후 타입 고정)
- 오름차순/내림차순 자동 감지 반복문

### 프로그래밍 경험자를 위한 한 줄 요약

```
변수 숫자 x = 10        →  int x = 10
출력(x)                 →  print(x)
조건 [x > 5] < ... ~>  →  if (x > 5) { ... }
반복 i [1~10] < ... ~> →  for i in range(1, 11): ...
함수 더하기 인수 a, b < 반환 a + b ~>  →  def 더하기(a, b): return a + b
```

## 2. 설치 및 시작

### 요구 사항

- Python 3.10 이상

### 설치

```bash
git clone https://github.com/nslookup255-pixel/Hanco-lang
cd Hanco-lang
pip install .
```

### CLI 명령어

```bash
hanco run <파일>      # .hanco 파일 실행
hanco repl            # 대화형 REPL 실행
hanco version         # 버전 확인
hanco <파일>.hanco    # 파일 직접 실행 (단축 형태)
```

### 파일 실행

확장자 `.hanco`로 파일을 만들고 실행합니다.

```bash
hanco run hello.hanco
# 또는
hanco hello.hanco
```

### REPL (대화형 모드)

```bash
hanco repl
```

```
한코 REPL v0.1.0
종료하려면 exit 입력
>>> 출력("안녕하세요!")
안녕하세요!
>>> exit
```

### Hello, World!

`hello.hanco` 파일을 만들고 아래 내용을 작성합니다.

```
출력("안녕하세요, 한코!")
```

```bash
hanco run hello.hanco
:: 출력: 안녕하세요, 한코!
```

## 3. 기본 문법 규칙

### 블록 `< ~>`

한코에서 블록은 `<`로 열고 `~>`로 닫습니다.

```
조건 [조건식] <
    :: 여기가 블록
~>
```

### 함수 호출 `()`

함수 호출은 `()`를 사용합니다.

```
출력("안녕")
더하기(3, 5)
```

### 줄바꿈

한코는 줄바꿈을 문장의 끝으로 인식합니다. 세미콜론(`;`)이 없습니다.

```
출력("첫 번째")
출력("두 번째")
```

### 대소문자 구분

식별자는 대소문자를 구분합니다. `이름`과 `이름2`는 다른 변수입니다.

## 4. 출력과 입력

### 출력

```
출력(값)
출력(값1, 값2, 값3)
```

여러 값을 쉼표로 구분하면 공백으로 이어서 출력합니다.

```
출력("이름:", "홍길동")
:: 출력: 이름: 홍길동

변수 숫자 나이 = 20
출력("나이는", 나이, "살입니다.")
:: 출력: 나이는 20 살입니다.
```

### 입력

```
변수 문자열 이름 = 입력("이름을 입력하세요: ")
변수 숫자 나이 = 입력("나이를 입력하세요: ")
```

> **참고:** 변수 자료형이 `숫자` 또는 `실수`이면 입력값을 자동으로 변환합니다.

## 5. 변수와 자료형

### 선언 문법

```
변수 [자료형] [변수명] = [값]
```

### 자료형 종류

| 자료형 | 설명 | 예시 |
|--------|------|------|
| `숫자` | 정수 | `42`, `-7` |
| `실수` | 부동소수점 | `3.14`, `-0.5` |
| `문자열` | 문자열 (`문자`로도 선언 가능) | `"안녕"`, `'hello'` |
| `참거짓` | 불리언 | `참`, `거짓` |
| `목록` | 리스트 (별도 문법 사용) | `("사과", "바나나")` |
| `자유` | 타입 무관 (함수 매개변수에서 사용) | — |

### 예시

```
변수 숫자 나이 = 25
변수 실수 키 = 175.5
변수 문자열 이름 = "홍길동"
변수 문자 별명 = "길동이"   :: 문자 = 문자열과 동일
변수 참거짓 활성화 = 참
```

### 타입 고정

한 번 선언한 변수는 자료형이 고정됩니다.

```
변수 숫자 x = 10
x = 20       :: 정상
x = "문자"   :: 오류! 숫자 변수에 문자열 대입 불가
```

### 값 변경 (재대입)

선언 이후에는 `변수` 키워드 없이 대입합니다.

```
변수 숫자 점수 = 80
점수 = 95
출력(점수)   :: 95
```

## 6. 연산자

### 산술 연산자

| 연산자 | 의미 | 예시 |
|--------|------|------|
| `+` | 더하기 | `1 + 2` → `3` |
| `-` | 빼기 | `5 - 3` → `2` |
| `*` | 곱하기 | `4 * 3` → `12` |
| `/` | 나누기 | `7 / 2` → `3.5` |
| `%` | 나머지 | `7 % 2` → `1` |

### 비교 연산자

| 연산자 | 의미 | 예시 |
|--------|------|------|
| `==` | 같다 | `x == 10` |
| `!=` | 다르다 | `x != 10` |
| `<` | 미만 (작다) | `x < 10` |
| `>` | 초과 (크다) | `x > 10` |
| `<=` | 이하 (작거나 같다) | `x <= 10` |
| `>=` | 이상 (크거나 같다) | `x >= 10` |

### 논리 연산자

| 연산자 | 의미 | 예시 |
|--------|------|------|
| `그리고` | AND | `x > 0 그리고 x < 10` |
| `또는` | OR | `x == 0 또는 x == 1` |

### 문자열 연결

`+` 연산자는 피연산자 중 하나가 문자열이면 자동으로 문자열 연결을 수행합니다.

```
출력("나이: " + 25)     :: 나이: 25
출력("안녕" + "하세요")  :: 안녕하세요
```

## 7. 조건문

### 기본 형태

```
조건 [조건식] <
    :: 참일 때 실행
~>
```

### if / else

```
조건 [점수 >= 60] <
    출력("합격")
~>
아니면 <
    출력("불합격")
~>
```

### if / else if / else

`아니면 [조건]` 형태로 연속 조건을 작성합니다.

```
조건 [점수 >= 90] <
    출력("A")
~>
아니면 [점수 >= 80] <
    출력("B")
~>
아니면 [점수 >= 70] <
    출력("C")
~>
아니면 <
    출력("F")
~>
```

### 복합 조건

```
조건 [나이 >= 18 그리고 나이 <= 65] <
    출력("근로 가능 연령")
~>

조건 [색 == "빨강" 또는 색 == "파랑"] <
    출력("선택된 색")
~>
```

## 8. 반복문

### for 반복 (범위 반복)

```
반복 변수명 [시작~끝] <
    :: 반복할 코드
~>
```

시작과 끝을 포함하며, 시작 > 끝이면 자동으로 내림차순으로 동작합니다.

```
:: 오름차순 (1, 2, 3, 4, 5)
반복 i [1~5] <
    출력(i)
~>

:: 내림차순 (5, 4, 3, 2, 1)
반복 i [5~1] <
    출력(i)
~>
```

### while 반복 (조건 반복)

```
반복 [조건식] <
    :: 조건이 참인 동안 반복
~>
```

```
변수 숫자 n = 1
반복 [n <= 5] <
    출력(n)
    n = n + 1
~>
```

### 멈춤 / 건너뛰기

다른 언어의 `break` / `continue`에 해당합니다.

```
반복 i [1~10] <
    조건 [i == 5] <
        멈춤        :: 반복 종료
    ~>
    조건 [i % 2 == 0] <
        건너뛰기    :: 다음 반복으로
    ~>
    출력(i)
~>
:: 출력: 1 3
```

## 9. 목록 (리스트)

### 선언

```
변수 목록 변수명 = (값1, 값2, 값3)
```

```
변수 목록 과일 = ("사과", "바나나", "딸기")
변수 목록 숫자들 = (1, 2, 3, 4, 5)
```

### 인덱싱 (0부터 시작)

```
출력(과일[0])   :: 사과
출력(과일[2])   :: 딸기
```

### 값 변경

```
과일[0] = "포도"
출력(과일[0])   :: 포도
```

### 길이

```
변수 숫자 개수 = 길이(과일)
출력(개수)   :: 3
```

### 목록 메서드

| 메서드 | 설명 | 예시 |
|--------|------|------|
| `:추가(값)` | 끝에 항목 추가 | `과일:추가("포도")` |
| `:제거()` | 마지막 항목 제거 | `과일:제거()` |
| `:제거(인덱스)` | 특정 인덱스 항목 제거 | `과일:제거(0)` |
| `:자르기(시작~끝)` | 부분 목록 반환 | `과일:자르기(0~2)` |

```
변수 목록 목록 = (10, 20, 30)
목록:추가(40)
출력(목록)          :: (10, 20, 30, 40)

목록:제거()
출력(목록)          :: (10, 20, 30)

변수 목록 부분 = 목록:자르기(0~2)
출력(부분)          :: (10, 20)
```

## 10. 함수

### 선언

```
함수 [함수명] 인수 [매개변수1, 매개변수2, ...] <
    :: 함수 본문
    반환 [값]
~>
```

매개변수가 없는 경우:

```
함수 인사 인수 <
    출력("안녕하세요!")
~>
```

### 호출

```
함수명(인자1, 인자2)
```

### 예시

```
함수 더하기 인수 a, b <
    반환 a + b
~>

변수 숫자 결과 = 더하기(3, 5)
출력(결과)   :: 8
```

### 재귀 함수

```
함수 팩토리얼 인수 n <
    조건 [n <= 1] <
        반환 1
    ~>
    반환 n * 팩토리얼(n - 1)
~>

출력(팩토리얼(5))   :: 120
```

### 스코프 규칙

함수 안에서 선언한 변수는 함수 내부에서만 유효합니다. 전역 변수는 함수 안에서 읽을 수 있습니다.

```
변수 숫자 전역값 = 100

함수 테스트 인수 <
    출력(전역값)      :: 100 (전역 변수 읽기 가능)
    변수 숫자 지역값 = 200
    출력(지역값)      :: 200
~>

테스트()
:: 출력(지역값)   :: 오류! 함수 밖에서 지역 변수 접근 불가
```

## 11. 메서드

메서드는 `:메서드명(인자)` 형태로 호출합니다.

### 문자열 메서드

| 메서드 | 설명 | 예시 |
|--------|------|------|
| `:포함(문자열)` | 부분 문자열 포함 여부 | `문장:포함("안녕")` → `참`/`거짓` |
| `:나누기(구분자)` | 구분자로 분리해 목록 반환 | `문장:나누기(",")` |
| `:제거앞뒤공백()` | 앞뒤 공백 제거 | `문장:제거앞뒤공백()` |
| `:자르기(시작~끝)` | 부분 문자열 반환 | `문장:자르기(0~3)` |

```
변수 문자열 문장 = "안녕, 한코!"

출력(문장:포함("한코"))          :: 참
출력(문장:자르기(0~2))           :: 안녕

변수 문자열 데이터 = "a,b,c"
변수 목록 항목들 = 데이터:나누기(",")
출력(항목들)                     :: (a, b, c)

변수 문자열 공백있음 = "  한코  "
출력(공백있음:제거앞뒤공백())    :: 한코
```

## 12. 자료형 변환

자료형 이름을 함수처럼 사용해 변환합니다.

```
숫자(값)    :: 정수로 변환
실수(값)    :: 실수로 변환
문자열(값)  :: 문자열로 변환
참거짓(값)  :: 불리언으로 변환
```

```
변수 숫자 정수 = 숫자("42")
출력(정수)          :: 42

변수 실수 소수 = 실수(7)
출력(소수)          :: 7.0

변수 문자열 글 = 문자열(3.14)
출력(글)            :: 3.14

변수 참거짓 판단 = 참거짓(1)
출력(판단)          :: 참
```

## 14. 전체 예제

### 예제 1: 피보나치 수열

```
함수 피보나치 인수 n <
    조건 [n <= 1] <
        반환 n
    ~>
    반환 피보나치(n - 1) + 피보나치(n - 2)
~>

반복 i [0~9] <
    출력(피보나치(i))
~>
```

### 예제 2: 성적 계산기

```
변수 숫자 점수 = 숫자(입력("점수를 입력하세요: "))

조건 [점수 >= 90] <
    출력("등급: A")
~>
아니면 [점수 >= 80] <
    출력("등급: B")
~>
아니면 [점수 >= 70] <
    출력("등급: C")
~>
아니면 [점수 >= 60] <
    출력("등급: D")
~>
아니면 <
    출력("등급: F")
~>
```

### 예제 3: 목록 처리

```
변수 목록 이름들 = ("철수", "영희", "민준")
변수 숫자 총인원 = 길이(이름들)

출력("총 인원:", 총인원)

반복 i [0~총인원 - 1] <
    출력(i + 1, "번:", 이름들[i])
~>
```

### 예제 4: 소수 판별

```
함수 소수판별 인수 n <
    조건 [n < 2] <
        반환 거짓
    ~>
    변수 숫자 i = 2
    반복 [i * i <= n] <
        조건 [n % i == 0] <
            반환 거짓
        ~>
        i = i + 1
    ~>
    반환 참
~>

반복 i [2~20] <
    조건 [소수판별(i)] <
```

## 16. 에러 메시지 해석

한코의 에러 메시지는 모두 한국어로 출력됩니다.

| 메시지 | 원인 | 해결 방법 |
|--------|------|-----------|
| `선언하지 않은 변수 'X'를 찾을 수 없습니다.` | 변수 선언 전 사용 | `변수 ... X = ...` 로 먼저 선언 |
| `변수 'X'에는 숫자 자료형만 저장할 수 있습니다.` | 타입 불일치 대입 | 자료형에 맞는 값 대입 또는 변환 함수 사용 |
| `정의하지 않은 함수 'X' 입니다.` | 함수 선언 전 호출 | 함수를 먼저 선언하거나 이름 확인 |
| `멈춤은 반복문 안에서만 사용할 수 있습니다.` | 반복문 밖에서 `멈춤` 사용 | `반복` 블록 안으로 이동 |
| `목록 인덱스 범위를 벗어났습니다.` | 없는 인덱스 접근 | `길이(목록)` 으로 범위 확인 후 접근 |
| `[N번 줄] 'X' 필요 (현재: Y)` | 문법 오류 | N번 줄 문법 확인 |
